# HireGrid — Job Application Tracker

A full-stack MERN application for tracking job applications, 
managing recruiter contacts, and organizing your job search.

## Tech Stack
- Frontend: React, Tailwind CSS, shadcn/ui
- Backend: Node.js, Express
- Database: MongoDB Atlas
- Auth: Clerk
- Deployment: Vercel (client) + Render (server)

## Features
- User authentication with Clerk
- Track job applications with status updates
- Manage recruiter/hiring manager contacts
- Add notes per application
- Dashboard with application stats
- Filter and search applications
